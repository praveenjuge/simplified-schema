# Simplified Schema

> A simple copy-pastable version of [Schema.org](https://schema.org/)'s content.

## Available Schemas

- Article
- Breadcrumbs
- Event
- Job Posting
- Local Business
- Organization
- Person
- Product
- Recipe
- Website

## Quickstart

First, you should install Hugo,

```
brew install hugo
```

Run the site using this command,

```
hugo server
```

Find out more about [Hugo here](https://gohugo.io/about/)
